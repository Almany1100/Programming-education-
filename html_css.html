مرحبًا بك في دورة HTML وCSS

ما هي HTML؟

HTML (لغة ترميز النصوص التشعبية) هي اللغة الأساسية لبناء صفحات الويب. تُستخدم لإنشاء هيكل الصفحة وتحديد العناصر مثل العناوين، الفقرات، الروابط، الصور، وغيرها.

أمثلة على عناصر HTML:

<h1>هذا عنوان رئيسي</h1>
<p>هذه فقرة نصية.</p>
<a href="https://example.com">رابط</a>
<img src="image.jpg" alt="وصف الصورة">

ما هي CSS؟

CSS (أوراق الأنماط المتتالية) هي اللغة المستخدمة لتنسيق عناصر HTML. تُستخدم لإضافة الألوان، الخطوط، المسافات، والتخطيطات إلى الصفحة.

أمثلة على CSS:

body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
}

h1 {
    color: #2196F3; /* لون أزرق */
    text-align: center;
}

p {
    font-size: 16px;
    line-height: 1.5;
}
